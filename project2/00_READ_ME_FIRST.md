## 🚀 How to Use

### Get Started in 3 Steps:

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev

# 3. Open browser to http://localhost:5173
```

### Customize in 2 Steps:

1. **Edit data in:** `src/pages/TalbeenaLanding.jsx`
   - Products array
   - Benefits array
   - Navigation links
   - Company info

2. **Edit colors in:** `src/components/*/Component.css`
   - Find `#2d5016` for primary color
   - Find `#f5deb3` for accent color

### Deploy in 1 Step:

```bash
npm run build  # Creates optimized dist/ folder
```
